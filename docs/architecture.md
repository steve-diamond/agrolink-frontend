# Architecture

## Overview
Agrolink frontend is built with Next.js 15, React, and Tailwind CSS. It uses the App Router and is structured for scalability and maintainability.

## Main Folders
- `app/`: Main application routes and pages
- `public/`: Static assets
- `styles/`: Global and component styles

## Key Technologies
- Next.js (App Router)
- React
- Tailwind CSS
- TypeScript

## Component Structure
- Modular, reusable components in `components/`
- Feature-based folders in `app/`

## Data Flow
- API calls to backend (Node.js/Express)
- Client-side state managed with React hooks
