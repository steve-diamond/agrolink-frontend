# UI Wireframes (Textual Description)

## Farmers

- Dashboard: Overview of sales, wallet, and notifications
- Produce Management: List, add, edit, and remove produce
- Loan Application: Form to request loans, list of loan statuses
- Orders: List of orders from buyers
- Wallet: Balance and transaction history

## Buyers

- Marketplace: Browse/search/filter produce, add to cart
- Checkout: Order summary and payment
- Orders: List of past/current orders, status tracking

## Drivers

- Delivery Dashboard: Assigned deliveries, status update buttons
- Map View: Route and delivery locations

## Admins

- Admin Panel: User management (approve, suspend, remove)
- Analytics: Charts for sales, users, orders, loans
- Content Moderation: Review listings, feedback, disputes

## Agents

- Onboarding Dashboard: Register new farmers, upload documents
- Progress Tracker: List of onboarding requests, status updates
- Support: FAQ and help resources
